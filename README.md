### 开发环境
> 使用VS2019开发
> 
> 依赖`.NET4.0`

### 运行环境
- 需要`.NET4.0`运行时环境；
- `64位Windows`电脑运行。

### 安装和使用
- 解压到`不含空格和中文`的目录；
- 运行`AdbTools.exe`程序，开始使用。

### 设备配对
- 输入框中输入安卓设备`使用配对码配对设备`提供的的`IP:Port`，在弹出的cmd窗口中输入`配对码`完成配对。

### 设备连接
- 输入框中输入安卓设备`无线连接`的`IP:Port`，支持省略`Port`只输入`IP`，此时端口默认为`5555`；
- 点击`连接设备`，等待连接成功。

### 安装apk
- 右键`已连接列表`中的设备`IP:Port`形式的地址，点击`安装应用`选择需要安装的`apk`程序，等待完成安装。
